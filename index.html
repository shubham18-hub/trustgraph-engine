📡 Live API Endpoints

Test these endpoints directly in your browser or via API calls. All endpoints are production-ready and fully functional.

GET /health

Check system health and service status

POST /api/v1/workers/register

Register a new informal worker using voice-first AI KYC and generate a blockchain identity

GET /demo/worker/{id}

Retrieve a verified worker profile, including skill tags, spoken languages, and dynamic trust score

POST /api/v1/blockchain/verify-credential

Verify an employment or skill credential against the TrustGraph ledger

GET /api/v1/analytics/demographics

Fetch aggregated, anonymized data on informal worker distribution across 22 states
