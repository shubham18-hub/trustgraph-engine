OPTION 1: Install Standard Python (RECOMMENDED)
------------------------------------------------

1. Download Python 3.11 or 3.12 from:
   https://www.python.org/downloads/

2. During installation:
   ✓ Check "Add Python to PATH"
   ✓ Check "Install pip"

3. Open NEW PowerShell window and run:
   python -m pip install -r requirements.txt

4. Start application:
   python app.py

5. Open browser:
   http://localhost:8000

OPTION 2: Install Rust for MSYS2 (ADVANCED)
--------------------------------------------

If you want to keep MSYS2 Python:

1. Install Rust:
   pacman -S mingw-w64-ucrt-x86_64-rust

2. Then install packages:
   .\venv\bin\pip.exe install -r requirements.txt

3. Start with venv Python:
   .\venv\bin\python.exe app.py

OPTION 3: Use Docker (EASIEST)
-------------------------------

1. Install Docker Desktop

2. Build image:
   docker build -t trustgraph .

3. Run container:
   docker run -p 8000:8000 trustgraph

4. Open browser:
   http://localhost:8000
