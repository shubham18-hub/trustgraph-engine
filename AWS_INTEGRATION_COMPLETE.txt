================================================================================
QUICK START - AWS INTEGRATION
================================================================================

Step 1: Install AWS CLI
  Windows: Download from https://awscli.amazonaws.com/AWSCLIV2.msi
  Linux/Mac: curl + install script

Step 2: Configure AWS Credentials
  Command: aws configure
  Enter: Access Key ID, Secret Key, Region (ap-south-1)

Step 3: Test AWS Connection
  Command: test-aws-connection.bat
  Expected: All 12 tests pass

Step 4: Enable Bedrock (AI Models)
  1. Go to AWS Console > Bedrock > Model access
  2. Request access to Claude models
  3. Wait for approval (usually instant)

Step 5: Create AWS Resources
  Command: aws-setup.bat
  Creates: S3, DynamoDB, KMS, IAM, CloudWatch, Secrets

Step 6: Update Configuration
  1. Review aws-config.env
  2. Update .env.production with AWS values
  3. Add API keys (Bhashini, Aadhaar, UPI)

Step 7: Deploy Application
  Command: DEPLOY_PRODUCTION.bat
  Select: AWS CloudFormation option
