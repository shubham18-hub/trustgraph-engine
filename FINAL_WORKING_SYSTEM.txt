================================================================================
  HOW TO USE
================================================================================

1. START THE SERVER
   
   Method 1 (Recommended):
   > START_APP.bat
   
   Method 2 (Manual):
   > python app.py

2. OPEN IN BROWSER
   
   Landing Page: http://localhost:8000
   
   This will show a beautiful landing page with:
   - TrustGraph branding
   - Statistics (490M workers, 22 languages, 100% secure)
   - "Get Started" button
   - Language switcher in top-right

3. SIGN UP / LOGIN
   
   Click "Get Started" → Redirects to /auth.html
   
   For Signup:
   - Click "नया खाता बनाएं" (Create New Account)
   - Fill in:
     * Name (required)
     * Phone: 10 digits (required)
     * Aadhaar: 12 digits (required)
     * Email, City, State, Pincode (optional)
   - Click "साइन अप करें" (Sign Up)
   - OTP will be shown on screen (e.g., "OTP: 123456")
   - Enter the OTP
   - Click "सत्यापित करें" (Verify)
   - Redirected to dashboard

   For Login:
   - Enter phone number
   - Click "OTP भेजें" (Send OTP)
   - OTP shown on screen
   - Enter OTP
   - Click "सत्यापित करें" (Verify)
   - Redirected to dashboard

4. USE DASHBOARD
   
   After login, you'll see:
   - Your name and profile
   - Trust score (720 by default)
   - Voice command button
   - Quick actions: Credentials, Jobs, Payments, Profile
   - Bottom navigation: Home, Work, Wallet, Settings
   - Language switcher
   - Theme switcher

5. CHANGE LANGUAGE
   
   - Click language button in top-right (shows flag + language name)
   - Select from 10 languages
   - All text updates instantly
   - Language preference saved

================================================================================
  TECHNICAL DETAILS
================================================================================

Backend:
  - Framework: FastAPI
  - Language: Python 3.11
  - Database: SQLite (trustgraph.db)
  - AI: AWS Bedrock (Claude 3 Haiku)
  - Authentication: JWT tokens (60 min expiry)
  - OTP: 6 digits, 5 min expiry, max 3 attempts

Frontend:
  - Pure HTML/CSS/JavaScript (no frameworks)
  - Responsive design (mobile-first)
  - Accessibility: WCAG AA compliant
  - Themes: 6 color schemes
  - Languages: 10 Indian languages
  - Voice: Web Speech API

Security:
  - Aadhaar: SHA-256 hashed
  - Passwords: Not stored (OTP-based auth)
  - Tokens: JWT with expiration
  - Headers: Security headers on all responses
  - Rate limiting: 60 requests/minute
  - Audit logging: All requests logged

Database Schema:
  - users: 20+ fields (name, phone, email, city, etc.)
  - otp_codes: phone, otp, expiry, attempts
  - sessions: session_id, user_id, token, expiry
  - credentials: work credentials and ratings
  - transactions: payment history
  - trust_scores: calculated trust scores

================================================================================
  TROUBLESHOOTING
================================================================================

Server won't start?
  - Check if port 8000 is in use
  - Run: netstat -ano | findstr :8000
  - Kill process if needed

Can't login?
  - Make sure you signed up first
  - Use the exact phone number you signed up with
  - OTP is shown on screen - copy it exactly
  - OTP expires in 5 minutes

Dashboard not showing?
  - Open browser console (F12)
  - Check for JavaScript errors
  - Clear browser cache (Ctrl + Shift + Delete)
  - Try incognito/private mode

Language not changing?
  - Make sure i18n.js is loaded
  - Check browser console for errors
  - Try refreshing the page

Database errors?
  - Delete trustgraph.db
  - Restart server (it will recreate)
